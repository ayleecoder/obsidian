### 문제 설명 
정수 `n`이 주어질 때, `n`이하의 짝수를 모두 더한 값을 return 하도록 solution 함수를 작성해주세요.

##### 제한사항
0 < n ≤1000

---------------
### 풀이 과정 
```js
function solution(n) {
    var answer = 0;
    for( let i = 1; i < n + 1; i++){
        if (i % 2 == 0){
            answer += i 
        }
    }
    return answer;
}
```
n은 0 보다 큰 정수이므로 i는 1 부터 시작하고 n 이하의 짝수이므로 주어진 정수값 + 1 보다 작은 정수이자 짝수 

짝수는 2로 나눌 때 나머지가 0 인 값들
